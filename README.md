# AI 寶可夢短篇小說產生器

本專案是一個創新的 AI 驅動寶可夢故事生成器，運用先進的 Chain-of-Thought（CoT）推理技術，讓使用者能夠輕鬆創作高品質的中文寶可夢短篇小說。

> **推薦使用 Colab 版本**：零設定、免安裝，立即體驗完整功能！  
> **開發者選擇完整版**：模組化架構、完整開發環境、API 整合支援！

## 主要功能

### 智能 CoT 推理引擎
- **11 階段完整 CoT 流程**：從故事規劃到最終審查
- **智能審查與修訂系統**：自動優化故事品質
- **多元化輸入處理**：主題、類型、寶可夢、概要等

### 進階功能
- **輸入優化建議**：協助改善故事設定
- **概要擴展建議**：豐富故事內容
- **角色深度塑造**：詳細角色檔案生成
- **場景細緻描述**：沉浸式環境建構
- **情節轉折建議**：增加故事張力
- **風格語氣調整**：個性化寫作風格
- **故事分支探索**：多元發展可能

### 豐富資料庫
- **99+ 隻經典寶可夢**：涵蓋第1-4世代
- **完整多語言支援**：中文、英文、日文名稱
- **RAG 增強功能**：智能寶可夢知識檢索

### 使用者介面
- **直觀的 Gradio 介面**：簡潔易用的網頁操作
- **即時故事生成**：所見即所得的創作體驗
- **10+ 精選範例**：快速入門創作
- **完整下載功能**：大綱和故事皆可儲存

### 方案二：完整版本（推薦開發者）
適合需要客製化、進階開發和本地部署的使用者。

#### 本地安裝與啟動
1. **Clone專案**：
   ```bash
   git clone https://github.com/your-username/pokemon-story-generator.git
   cd pokemon-story-generator
   ```

2. **安裝依賴**（建議使用虛擬環境）：
   ```bash
   # 使用 Poetry（推薦）
   poetry install
   
   # 或使用 venv
   python -m venv env
   env\Scripts\activate  # Windows
   # source env/bin/activate  # macOS/Linux
   pip install -r requirements.txt
   ```

3. **設定 OpenAI API Key**：
   - 在專案根目錄建立 `.env` 檔，內容：
     ```
     OPENAI_API_KEY=sk-your-openai-api-key-here
     ```

4. **啟動 Gradio 介面**：
   ```bash
   python -m ui.app
   # 或直接執行
   python ui/app.py
   ```

5. **訪問應用程式**：
   - 打開瀏覽器訪問：`http://localhost:7860`

### 選擇指南

#### **選擇 Colab 版本**，如果您：
- **新手用戶**：剛開始接觸 AI 故事生成
- **快速體驗**：想立即試用所有功能
- **教學演示**：需要向他人展示系統
- **偶爾使用**：不需要頻繁或大量使用
- **零技術門檻**：不想處理安裝和配置
- **分享簡單**：需要輕鬆分享給他人
- **學術研究**：用於課程或研究展示

#### **選擇完整版本**，如果您：
- **專業開發**：需要修改或擴展功能
- **生產環境**：要部署到正式服務中
- **頻繁使用**：需要長期穩定的環境
- **批次處理**：要自動化大量故事生成
- **API 整合**：需要與其他系統整合
- **客製化需求**：要根據特殊需求修改
- **企業應用**：需要企業級的可靠性
- **離線使用**：需要在無網路環境運行

### 版本遷移

**從 Colab 升級到完整版**：
1. 複製 Colab 中的自定義設定
2. 按照完整版安裝步驟設定環境  
3. 使用相同的 API 金鑰和參數
4. 享受更多進階功能！

**從完整版簡化到 Colab**：
1. 將重要的故事和設定儲存
2. 打開 Colab 版本 notebook
3. 導入您的設定和偏好
4. 繼續享受無縫體驗！
